# Reader-Movies
### 功能模块
- 电影排行版
- 电影详细内容
- 搜索

### 技术要点
- 微信小程序组件
- 微信小程序事件跳转
- 微信小程序template使用
- 豆瓣开放API
- flex弹性盒子模型
